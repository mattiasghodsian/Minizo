# Features

- browsing by folder.
- managing files.
- grab youtube video and convert to audio file.


# Requirements
- [Node.js](https://nodejs.org/en)
- [youtube-dl](https://github.com/ytdl-org/youtube-dl)
- [ffmpeg](https://ffmpeg.org/)
